# ili9327_rpi

ILI9327 TFT Library for Raspberry


This is library for RaspberryPi/OrangePi.   
This library can show a chart to ILI9327 TFT Shield.   

----

Wirering   
None   

build   
cc -o demo demo.c fontx.c ili9327.c -lwiringPi -lm
